This is your first blog post! You can write content in **Markdown** and even use React components if needed.

### Features

- Write in Markdown
- No external CMS required
- Version controlled with Git
- Free forever

### Getting Started

To create a new blog post, simply add a new `.mdx` file to the `content/posts` directory with frontmatter like this one.
